📢 어머님께서 늘 꼼꼼하게 채점해 주시고 아이가 풀 때 어떤 상황이었는지(급하게 풀었는지 등) 코멘트까지 남겨주셔서, 우리 아이가 집에서 어떻게 공부하고 있는지 파악하는 데 정말 큰 도움이 되었습니다. 한명 한명 세심하게 보면서 거의 논문 수준이 되다 보니 이제 데이터가 점점 쌓이고 있네요. 이제 더 명확하고 빠르게 진행될 것 같습니다. 지금은 제가 손으로 하나하나 보면서 설명 드리고 있어서 수업 이상의 힘이 들어가거든요..... 저도 제가 지도한 다른 방향으로 과제 해올 떄 보면 제가 더 죄송 스럽기도 하네요.
✅ 과제 확인
GM13 유사문제
✅ 수행
[필수] 주한이 주고받기 어려워 했던 유형 유사 문항입니다. 문제 1, 2, 3 ) 우선 4,5,6에 써 놨지만 차이의 절반 기억삭제! 주한이는 아직 공식화 하기에는 혼란스러움이 많습니다. 조금씩 실력을 키워가며 유형을 통해 깨우쳐서 공식화가 되어야 겠어요. 오히려 공식을 그냥 외워서 풀다 보면 나중에 문제가 조금만 꼬여도 당황하고 실수할 수 있거든요. 그런데 부모님께 잘 보이고 싶은 마음이 있으니(가끔 저한테도...) 더 마음이 좁아지고 실수가 더 많은 것 같아요. 우리 엄마 앞에서는 작아질때도 있잖아요. 그래서 당분간은 공식을 머릿속으로 계산하기보다, 제가 그려준 것처럼 귤이나 사과 동그라미를 직접 그려서 짝을 맞춰 옮겨보는 시각화 훈련을 하는 것이 훨씬 확실한 학습법이라 생각이 됩니다. 문제 7, 8, 9) 어머님께서 남겨주신 메모 중 "처음에 급하게 읽으며..." 부분에 제가 파란색으로 동그라미 쳐둔 것 보셨죠? 사실 주한이 뿐만 아니라 모든 아이들이 다 급합니다. 자주 놓쳐요. 연산을 패턴화 해서 그런것도 있지만 제가 보기에는 제한된 시간에 문제를 풀려다 보니 시간 여유가 많지는 않은듯 합니다.★ 표를 이용하여 해결하기는 너무 최고 GOOD~!!
더클래식_1과정_8N30권_합과 차의 배수 문제
✅ 수행
GM13 STEP1,2 빈문제
✅ 수행
처음에 준성이 공은 '네모(ㅁ)', 연우 공은 '세모(△)'로 서로 다른 기호를 써서 나름대로 똑똑하게 식을 세웠습니다. 그런데 제가 빨간 펜으로 "왜 모양이 다를까?"라고 코멘트를 달고 수정을 해주었는데요. 제가 알려준 방법이 아니거든요....나중에 중고등학교 때 배우는 방정식에서도 미지수(x, y 등)의 종류가 많아지면 많아질수록 계산이 복잡하고 풀기가 어려워지잖아요? 초등 과정의 문제 풀이도 마찬가지입니다. 동그라미, 세모, 네모 등 기호의 모양이 많아지면 아이들이 식을 세워놓고도 스스로 헷갈리게 됩니다. 그래서 기준이 되는 '가장 작은 수(연우)' 쪽에 동그라미(○) 하나를 두고, 큰 수(준성)는 그에 맞춰 동그라미 두 개(○○)로 통일해서 나타내도록 지도했습니다. 이렇게 종류를 하나로 줄여주면(○ 3개가 24니까 ○ 하나는 8) 훨씬 직관적이고 쉽게 문제를 해결할 수 있답니다. 방정식의 기초를 아주 제대로 다지는 과정이라고 보시면 됩니다! 기특한 건, 처음엔 기호를 여러 개 썼지만 다음 문제 풀때는 제가 알려준 방법을 기억해 낸것 같아요. 5번(포켓몬 카드), 6번 문제에서는 완벽하게 적용해서 풀었네요/ 동생을 ○ 하나로 두고 형을 ○ 5개에서 6을 빼는 식(○○○○○-6)으로 스스로 그려서 척척 풀어냈네요. 아직은 어리니 ○X6보다는 ○○○○○○으로 먼저 시작하는 것이 좋습니다. 지금은 -6과 5의 차이가 11이라는 개념이 더 중요해요. 그래도 칭찬에 약한 주한이니 폭푹 칭찬 해주세요^^
