02. 초간편 AI 학습 리포트

모든 것을 처음부터 입력할 필요가 없습니다. 시스템이 과거와 현재를 연결합니다.

로컬 파싱 (Local Parsing) 기술

이전에 PC에 저장된 리포트 HTML 파일을 클릭만 하면, 서버 통신 없이 0.1초 만에 이전 '오늘의 과제' 내역을 현재 리포트의 '과제 확인'란으로 자동 분류하여 불러옵니다.

음성 인식 & 멀티미디어 통합

마이크 버튼을 통한 실시간 음성-텍스트 변환, 유튜브 링크 삽입 시 썸네일 자동 추출 등 모바일 환경에서도 최적화된 리포트를 생성합니다.

알리고(Aligo) 즉시/예약 발송

AWS 서버를 경유하여 입력된 학생의 전화번호로 안내 메시지를 발송합니다. 클릭 한 번으로 내일 아침 9시로 예약 발송을 설정할 수 있습니다.

04. 보이지 않는 백엔드 (Invisible Backend)

프론트엔드에서 버튼 하나를 누를 때, 백엔드에서는 4가지 자동화 프로세스가 동시에 동작합니다.

구글 드라이브 & 대시보드 자동화

학생별 개인 폴더가 자동 생성되어 HTML 리포트가 저장됩니다. 동시에 구글 시트 17개 컬럼(Column)에 수강 과목 3개 분리, 뷰어 하이퍼링크 수식이 완벽히 매칭되어 기록됩니다.

노션 DB & 구글 캘린더 동기화

상담 예약 및 보강 일정이 확정되면 노션 데이터베이스와 구글 캘린더에 동시 등록됩니다. 만약 일정이 '변경'되면, 노션의 기존 과거 기록을 자동으로 아카이브(삭제) 처리하고 새 일정으로 갱신합니다.
